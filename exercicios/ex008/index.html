Principais formatações

Negrito / Destaque

Nesse parágrafo temos o um termo em negrito usando a tag B (NÂO SEMANTICA).

Já nesse outro parágrafo temos o termo em negrito usando a tag STRONG (SEMANTICA)

Aqui temos um termo em italico usando a tag I (não semantica)

Aqui ja o termo em italico usando a tag EM (semantica.)

Lorem ipsum dolor sit amet consectetur adipisicing elit. Nesciunt ipsam veniam fugit quis qui error quia, ullam voluptates ex tempore ut unde itaque laborum, a eos, tenetur inventore neque? Quidem.

Texto marcado

Podemos criar também um texto marcado, com a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas nao considerando. usando a TAG 'del'.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. usando a TAG 'ins' .

Temos também o sublinhado usando a TAG 'u' (não semântica)

Texto sobrescrito

Para inserir coisas tipo x2+3.

Texto subescrito

Para inserir coisa tipo H20.
